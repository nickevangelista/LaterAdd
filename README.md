# LaterAdd
LaterAdd é um aplicativo em Java para gerenciamento de listas pessoais de conteúdos para consumo futuro, como livros, filmes, séries e jogos.

O projeto foi desenvolvido com foco em praticar Programação Orientada a Objetos e persistência de dados em arquivos.

## Funcionalidades
- Adicionar livros, filmes, séries e jogos
- Listar itens cadastrados
- Persistência dos dados em arquivos locais
- Interface via terminal (CLI)

## Tecnologias utilizadas
- Java
- Programação Orientada a Objetos
- Manipulação de arquivos
- Collections (List)

## Estrutura do projeto
- `Main`: interface de interação com o usuário
- Classes de domínio: Livro, Filme, Serie, Jogo
- `ArquivoService`: responsável pela leitura e escrita dos dados

## Como executar
1. Compile o projeto
2. Execute a classe `Main`
3. Utilize o menu interativo no terminal
